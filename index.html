Vindo

Run Windows programs. On Mac.

Download

Vindo can run Windows games and programs on your Mac, without needing to boot into a separate system or launch a VM program. This means that Vindo will not waste resources in the way that virtual machines do, and won't require irritating boot times.

With Vindo, when you want to run an .exe, simply double-click it. The .exe will start, and you're good to go!
